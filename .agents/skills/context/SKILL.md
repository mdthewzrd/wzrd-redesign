# Context Skill

## Purpose
Core context management and optimization skill. Provides token optimization, context pruning/distillation, and memory management.

## Aliases & Redirects
This skill acts as a facade/aggregator for:
- `context-driven-development` - Project context management
- `context` (base skill) - Core context operations
- `memory-curation` - Long-term memory management
- `auto-memory` - Automatic memory injection
- `conductor` - Structured context framework

## When to Use
Use this skill for:
- Token optimization and budget management
- Context pruning and distillation
- Memory retention and retrieval
- Conversation state management
- Project context establishment
- Knowledge integration and synthesis

## Integration
- Automatically loaded by ALL blueprints for context optimization
- Works with rules-based context system for token allocation
- Integrated with validation pipeline for context quality checks
- Recommended by tool shed for all task types

## Context Optimization Strategies
1. **Mode-Based Optimization** - Different rules per task type
2. **Priority Retention** - Keep critical info, prune tangential
3. **Progressive Disclosure** - Reveal context as needed
4. **Distillation** - Extract essence from verbose content
5. **Pruning** - Remove noise and redundancy

## Token Allocation Rules
- **Coding**: 70% code, 20% structure, 10% explanation
- **Debugging**: 60% problem, 30% analysis, 10% solution  
- **Research**: 40% sources, 30% analysis, 20% synthesis, 10% conclusion
- **Planning**: 50% structure, 30% dependencies, 20% validation

## Context Management Operations
- **Prune** - Remove irrelevant/redundant context
- **Distill** - Extract key insights from verbose content
- **Retain** - Keep critical architecture/decisions
- **Optimize** - Apply mode-specific token allocation
- **Validate** - Ensure context quality and relevance

## Success Metrics
- Token usage within budget (> 80% efficiency)
- Context relevance to current task (> 90% relevant)
- Memory retention of critical information
- Quality of distilled knowledge
- User satisfaction with context management
