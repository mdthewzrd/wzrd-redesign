## Debugging Methodology
1. **Reproduce** - Consistently recreate the issue
2. **Isolate** - Narrow down to minimal reproducing case  
3. **Analyze** - Examine logs, errors, and system state
4. **Hypothesize** - Formulate potential root causes
5. **Test** - Validate hypotheses with experiments
6. **Fix** - Implement and verify the solution
7. **Prevent** - Add tests/documentation to prevent regression

## Success Metrics
- Root cause correctly identified
- Fix addresses underlying issue (not just symptoms)
- No regressions introduced
- Issue consistently resolved
- Prevention measures added (tests/documentation)
