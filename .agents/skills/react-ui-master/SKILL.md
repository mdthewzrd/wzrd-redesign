# React UI Master Skill

## Purpose
Master-level React UI/UX development skill. Provides advanced React patterns, component architecture, and UI/UX best practices.

## When to Use
Use this skill for:
- React component design and implementation
- UI/UX pattern implementation
- Design system creation and maintenance
- Performance optimization for React apps
- Accessibility implementation (WCAG compliance)
- Responsive design implementation
- Component library development
- Theme system implementation
- State management architecture
- React performance profiling and optimization

## Core React Patterns
1. **Component Architecture** - Container/presenter, compound components
2. **State Management** - Local state, context, Redux, Zustand, React Query
3. **Performance Optimization** - memo, useMemo, useCallback, virtualization
4. **Data Fetching** - SWR, React Query, suspense, streaming
5. **Composition Patterns** - Render props, children as function, hooks
6. **Testing Strategies** - React Testing Library, Jest, Cypress
7. **Accessibility** - ARIA attributes, keyboard navigation, screen readers

## Modern React Features
- **React 19** - Actions, directives, new hooks
- **Server Components** - Zero-bundle-size components
- **Concurrent Features** - startTransition, useDeferredValue
- **Suspense** - Data fetching with fallback UI
- **Streaming** - Progressive content loading
- **Optimistic Updates** - UI updates before server response

## Success Metrics
- Performance scores (Lighthouse > 90)
- Bundle size within targets (< 100KB initial load)
- Accessibility compliance (WCAG 2.2 AA)
- Mobile responsiveness (all screen sizes)
- Component reusability (> 70% reuse rate)
- Developer experience (easy to use/extend)
- Test coverage (> 80% component coverage)
