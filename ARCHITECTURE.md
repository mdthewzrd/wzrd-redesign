## Key Design Decisions

### 1. Separation of Concerns
- **Remi Core**: Agent identity and conversation
- **Conductor**: System orchestration
- **Tool Shed**: Capability library
- **Sandbox**: Project isolation

### 2. Multi-Interface Strategy
- Discord: Natural chat (primary)
- CLI: Terminal power users
- Web: Browser-based
- API: Programmatic access

### 3. Mode-Based Operation
Auto-switching based on request type:
- CHAT: Conversational
- CODER: Implementation
- THINKER: Design
- DEBUG: Problem-solving
- RESEARCH: Investigation

### 4. Proactive Engineering
- Suggests blueprints before you ask
- Validates quality automatically
- Optimizes context intelligently
- Anticipates next steps
